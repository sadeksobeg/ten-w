# Game-style assets — attribution

T.E.N.E.G.T.A Growth Engine uses a **world-class RPG / esports visual language** for badges, ranks, and avatar presets.

## Icon design source

- **game-icons.net** — [https://game-icons.net](https://game-icons.net)  
  Thousands of free SVG game icons (CC BY 3.0). Our inline icon paths in `site/lib/growth/game-icons/registry.ts` follow the same RPG/MMO vocabulary (trophy, crown, diamond, shield, etc.) and are optimized for TENEGTA’s gold/neon UI.

If you redistribute icons separately, credit **game-icons.net** and the original authors listed on that site.

## Avatar presets

- Preset portraits use custom SVG silhouettes (helm, crown, mage, etc.) with gradient frames — no third-party image files required.
- Optional future integration: [DiceBear](https://www.dicebear.com) (MIT) for procedurally generated avatars.

## UI references (design direction only)

- Glassmorphism + neon glow (Valorant / Riot-style HUD)
- Rank tiers: Bronze → Silver → Gold → Diamond → Legend
- Achievement badges: Common / Rare / Epic / Legendary rarities

## Not included (licensing)

We do **not** bundle paid asset packs (e.g. Craftpix) in the repository. Admins can upload custom images via partner profile settings.
