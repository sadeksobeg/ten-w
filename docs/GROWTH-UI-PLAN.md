## مبادئ التصميم (معايير عالمية)

1. **8pt grid** — هوامش 4/8/16/24/32px؛ لا قيم عشوائية.
2. **تسلسل بصري واحد** — عنوان → KPI → إجراء → تفاصيل.
3. **لا تراكب** — تنقل أفقي مع `overflow-x` + `scrollbar` مخفي؛ على الجوال: شريط سفلي أو قائمة «المزيد».
4. **أفاتار مميز** — تدرج لوني من البريد + حرفان؛ دعم `avatarUrl` لاحقاً (رفع صورة).
5. **RTL أولاً** — العربية مرجع؛ EN/FR مرآة منطقية.
6. **حالات فارغة/تحميل/خطأ** — لكل قائمة ولوحة.
7. **WCAG** — تباين 4.5:1، `focus-visible`، أزرار 44px لمس.

## المرحلة B — هيكل الشريك (PARTNER)

### B1 — Shell موحّد

```
┌─────────────────────────────────────────────┐
│ [Avatar] الاسم · المستوى    🔔  خروج      │
├─────────────────────────────────────────────┤
│ لوحة │ فعاليات │ محادثة │ (بروفايل عام)   │  ← scroll أفقي، لا wrap فوق بعض
├─────────────────────────────────────────────┤
│ KPI strip (4 بطاقات متساوية)                │
│ شريط نقاط القوة (XP)                        │
│ المحتوى (شبكة 12 عمود)                      │
└─────────────────────────────────────────────┘
```

- مكوّن `GrowthPartnerShell` يغلف كل صفحات `/growth/*` (ما عدا sign-in/register).
- شريط سفلي ثابت على `< md` (Dashboard / Events / Chat).

### B2 — لوحة التحكم

- صف KPI: أرباح | صفقات مفتوحة | ترتيب أسبوعي | سلسلة.
- تقليل طول الصفحة: أقسام قابلة للطي (مهام، شبكة، سجل عمولات).
- بطاقات صفقات: أفاتار عميل/منتج + شريط رحلة مضغوط.

### B3 — بروفايل عام

- صورة رأس كبيرة + شارة مستوى + QR (لاحقاً).
- روابط اجتماعية كأيقونات وليس نص طويل.

## المرحلة C — هيكل الأدمن (ADMIN)

### C1 — تنقل من مستويين (يحل التراكب)

```
الصف الأول:  نظرة عامة | صفقات | سحوبات | شركاء | محادثات
الصف الثاني: منتجات | شارات | مستويات | فعاليات | إشعارات | …
```

أو **Sidebar ثابت** على `lg+` و **Drawer** على الجوال (نمط Vercel/Stripe).

### C2 — لوحة نظرة عامة

- بطاقات KPI بأيقونة + رقم كبير + sparkline (اختياري).
- إجراءات سريعة: «إنشاء شريك»، «صفقة جديدة»، «إرسال إشعار».

### C3 — صفحات ثقيلة

- **شركاء:** جدول + أفاتار + فلاتر؛ لوحة جانبية للتفاصيل (لا modal فوق modal).
- **محادثات:** التصميم الحالي (3 أعمدة) — تحسين تباعد فقط.
- **فعاليات:** معاينة Markdown للقواعد.
